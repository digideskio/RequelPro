ReQueryApp
==========

Before Starting Development:

1. `npm install`
1. `cd src`
1. `bower install`
1. `cd ..`

During Development:

1. `grunt build` - Creates a timestamped build for Mac only
1. `grunt snapshot` - Creates a snapshot build for Mac only
1. `grunt dist` - Creates release builds for all platforms
