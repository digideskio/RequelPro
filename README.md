### How to I get it?

#### Download Binary

Not yet.

~~Binaries for Windows, OSX, & Linux can be found on the [Releases](https://github.com/jmdobry/RequelPro/releases) page.~~

#### Build From Source

1. `git clone git@github.com:jmdobry/RequelPro.git`
1. `cd RequelPro`
1. `npm install`
1. `npm run build`
1. Binaries will be built in `build/RequelPro/`
